## Project Overview

**FAFA (Fit Analysis For AI)** — A ClaudeCode-based script for parsing FIT files and preparing the data for AI analysis.

FIT (Flexible and Interoperable Data Transfer) is a binary file format used by Garmin and other sports devices to record workout/activity data (GPS tracks, heart rate, power, cadence, etc.).

Project is in early initialization. Only a README exists; no source code, build system, or test infrastructure has been added yet.
